Manual do Usuário - SPARK

1. Introdução

O SPARK é uma aplicação para monitoramento de consumo de energia elétrica. Com ele, você pode facilmente verificar a tensão, corrente e custo de energia de dispositivos conectados.

2. Conectando um Dispositivo

Para conectar um dispositivo, clique no botão "Conectar Dispositivo" na tela principal. Aguarde até que o sistema reconheça o dispositivo conectado.

3. Visualizando Informações

Após conectar um dispositivo, você verá informações sobre tensão elétrica, consumo, custo por hora, corrente e tensão do dispositivo.

4. Configurações

No menu de configurações, você pode:

  • Alternar entre modo claro e escuro
  • Mostrar ou ocultar tensão elétrica
  • Mostrar ou ocultar consumo
  • Mostrar ou ocultar custo por hora
